#### 说明

本工具适用 **CMCC-EDU** 的登录
绕过证书验证，实现内网登录（不保证都能正常登录）
并可通过后续的移动认证
主要方便部分没有网口的轻薄本用户和部分宽带接口无效的用户。
目前版本为 **1.1.0**
